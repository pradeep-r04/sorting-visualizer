# ✨ Sorting Visualizer

An interactive and animated Sorting Visualizer built using **HTML, CSS, and JavaScript**.  
This project demonstrates how different sorting algorithms work through real-time animations and sound visualization.

---

## 🚀 Live Features

- 🎨 Modern UI with smooth animations
- 🔊 Audio feedback during sorting
- 📊 Algorithm Complexity Display
- 🎚 Adjustable Array Size
- ⚡ Adjustable Sorting Speed
- 📱 Responsive Design
- 🌙 Beautiful gradient theme

---

## 🧠 Supported Algorithms

- Bubble Sort
- Insertion Sort
- Selection Sort
- Quick Sort
- Merge Sort

---

## 📈 Time Complexity

| Algorithm      | Best Case     | Average Case   | Worst Case     |
|--------------|--------------|--------------|--------------|
| Bubble Sort  | O(n)         | O(n²)        | O(n²)        |
| Insertion    | O(n)         | O(n²)        | O(n²)        |
| Selection    | O(n²)        | O(n²)        | O(n²)        |
| Quick Sort   | O(n log n)   | O(n log n)   | O(n²)        |

---

## 🎮 How It Works

1. Generate a random array.
2. Select a sorting algorithm.
3. Adjust size and speed if needed.
4. Click **Start Sorting**.
5. Watch the algorithm sort in real-time with visual and audio feedback.
